Slide Out Navigation

Like Stefan Matei pointed out, it might be quite cute but we don’t want the user to hover over all options first, in order to see what menu items exit.
So, I thought about labeling the menu items and making them stick out, so the user can always “see” what the navigation entails.
Being in a horizontal position makes everything readable and when the user hovers over the label, the rest of the item with the icon will slide out.


Although this tutorial is related to the previous one, I will start again from scratch – for the readers that are only interested in this new navigation.

I also changed the original icons and colors of this menu. So you can customize as you like.
